# backbonejs + Sails + MongoDB application

my first web application based on sailsjs
Install
------------------

    git clone https://github.com/muphy/p2c.git
    cd p2c
    npm install
    
You will want to configure your Mongo DB in config/adapters.js
    
    mongo: {
      module: 'sails-mongo',
      host: 'localhost',
      port: 27017,
      database: 'p2c'
    }
